# Meeting Notes Defaults And Clarification Prompts

Use these defaults only when the source material is incomplete and the user has not provided better information.

## Defaults

- Unknown location: `Not specified`
- Unknown meeting type: infer from context, otherwise `General meeting`
- Unknown action item status: `Open`
- Missing next meeting information: `Not scheduled`
- Missing attachments: `None provided`

Always record inferred details in `Assumptions / Gaps`.

## Clarify Before Drafting When Missing

Ask concise follow-up questions if any of these are missing and materially affect the output:

- Meeting title
- Date or time
- Participant list
- Decision owners or action item owners
- Deadlines for promised follow-up work
- Intended audience or distribution sensitivity

## Useful Clarification Prompts

- `Who attended, and what titles should be shown in the participant list?`
- `What decisions were finalized versus still under discussion?`
- `Who owns each follow-up task, and when is it due?`
- `Should anything be treated as confidential or excluded from the distributed minutes?`
- `Is there a next meeting already scheduled?`
