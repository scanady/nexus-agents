# Response Templates

## Standard Greeting Template

```
👋 Hello, {name}!

{message}

— Your friendly hello-world skill
```

### Field Reference

| Field | Required | Description |
|-------|----------|-------------|
| `{name}` | Yes | Recipient name. Use "World" or "there" if unknown. |
| `{message}` | Yes | A short, contextual sentence. Match the selected tone. |

### Tone Examples

**Casual**
```
👋 Hello, World!

Great to see you — hope your day is off to a solid start.

— Your friendly hello-world skill
```

**Professional**
```
👋 Hello, Dr. Smith!

Welcome — looking forward to working with you today.

— Your friendly hello-world skill
```

**Enthusiastic**
```
👋 Hello, Sean!

So glad you're here — let's make something awesome!

— Your friendly hello-world skill
```
